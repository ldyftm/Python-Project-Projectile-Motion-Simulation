# Python Project: Projectile Motion Simulation

A 3D interactive projectile motion simulation built with **Python** and **VPython**. Visualizes how launch angle and gravity affect projectile trajectory in real time.

## Objectives

1. Create a projectile motion simulation using Python and VPython
2. Visualize projectile motion and trajectory in real time
3. Calculate velocity, maximum height, horizontal range, and time of flight
4. Apply projectile motion physics concepts and formulas

---

## Simulations

| Simulation | Description |
|------------|-------------|
| **Simulation 1** | Different angles: 15°, 30°, 45°, 60°, 90° |
| **Simulation 2** | Gravity comparison: Earth (9.8), Moon (1.7), Mars (3.6) |
| **Simulation 3** | Offset launch: 15°, 30°, 45°, 60°, 75° from x = -4.5m |

---

## Features

- Launch and Reset buttons for each simulation
- 3D camera controls (rotate, pan, zoom)
- Colored trails showing projectile paths
- Console output showing time of flight and range

---

## Physics Formulas Used

- Horizontal velocity: vₓ = v₀·cos(θ)
- Vertical velocity: vᵧ = v₀·sin(θ)
- Time of flight: T = 2v₀·sin(θ)/g
- Horizontal range: R = v₀²·sin(2θ)/g
- Maximum height: H = v₀²·sin²(θ)/2g
